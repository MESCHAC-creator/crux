# 🎥 CRUX - APPLICATION DE VISIOCONFÉRENCE

Une application Flutter de vidéoconférence moderne et performante, meilleure que Zoom et Google Meet.

## ✨ Caractéristiques

- 🔴💜⚪ Design premium avec couleurs rouge, violet et blanc
- 🎥 Vidéoconférence haute définition via Agora.io
- 🔐 Authentification sécurisée Firebase
- 💾 Stockage cloud Firestore
- 📱 Interface responsive (mobile, web, desktop)
- 🌍 Multilingue (FR, EN, etc.)
- 🎨 UI/UX moderne avec animations fluides
- ⚡ Performance optimisée
- 🛡️ Gestion complète des erreurs

## 🚀 Installation Rapide

### 1. Cloner le projet
```bash
git clone https://github.com/yourusername/crux.git
cd crux
```

### 2. Installer les dépendances
```bash
flutter pub get
```

### 3. Configurer Firebase
```bash
flutter pub global activate flutterfire_cli
flutterfire configure --project=crux-8aa85
```

### 4. Lancer l'app
```bash
flutter run
```
